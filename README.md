html5-palette
=============


# 概覽 Overview

a simple online palette by nodejs &amp; HTML5

一個簡單的 HTML5 繪圖程序。

# 數據結構 DataStructure

    路徑格式{
        x, y, // 坐标
        drag, // 画笔是否中断
        tool, // 工具
        color, // 颜色
        size, // 筆刷大小
        alpha // 筆刷濃度
    }
